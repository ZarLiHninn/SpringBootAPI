## SpringBootAPIの使い方手順
1. リポジトリをクローンしてください。
   ```
   git clone https://github.com/ZarLiHninn/SpringBootAPI.git
   ```
2. UserData-dbのファイルをmysqlにインポートしてください。
3. 以下のコマンドで実行してください。
   ```
   ./mvnw spring-boot:run
   ```
4. localhost:8080/usersで実行してください。
